# VR-CyberDeck-Setup
A Step by Step Guide To Installing VR CyberDeck 

---

## ❓️ What is VR CyberDeck 
VR CyberDeck is a cross-platform desktop deck for managing, sideloading, and uploading content to Meta Quest devices. It plugs into a community-run game library.
Downloads and Installs any APK you choose.

---

## ⚠️ IMPORTANT Before you start!
If you haven't already, you need to get your headset and PC ready for sideloading.

## 🛡️ REAL TIME PROTECTION (Windows Defender Exclusion)
1. Open your **C: Drive**
2. Create a folder named: `RSL`
3. Inside `RSL`, create another folder named: `Rookie`
4. Open **Windows Security → Virus & Threat Protection → Manage Settings → Add or Remove Exclusions**
5. Add **Folder** exclusion: `C:\RSL\Rookie`
6. Click **APPLY**

> - If blocked: Right-click download → **Save Link As** → choose `C:\RSL\Rookie`

---

## 📦 INSTALLATION
1. Move **AndroidSideloader.exe** and **public.json** into: `C:\RSL\Rookie`
2. Run `AndroidSideloader.exe`
3. Select **PROVIDE PUBLIC CONFIG**
4. Browse and select the `public.json` file you just placed
5. Rookie will now connect to the updated server/database
6. Shortcut: Right-click `AndroidSideloader.exe` → **Create Shortcut** → move to Desktop

---

## 📦 HOW TO USE ROOKIE
1. Connect Headset To Computer
2. Put Heaset on
3. Allow `USB Debugging` and Check `Always allow` inside Headset
4. Run `Rookie` - Rookie will auto-detect device
5. Browse / Search for a Game
6. Choose a Game - Rookie will download and install

# INSTALLING

1. Install the build for your OS
2. Plug in your Quest via USB (data-capable cable)
3. Allow USB Debugging on the headset
4. Browse the library and hit download


## That's it. The bundled server config means there's nothing to configure on first run.
